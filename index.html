Fuente de datos 2026

Pega la URL de tu Google Sheet de 2026. Se guardará y el dashboard la usará para sincronizar automáticamente cada vez que abras el dashboard o presiones "Actualizar".

📋 Cómo obtener la URL (elige una opción): Opción A — Más fácil:
1. Abre el Sheet → Archivo → Compartir → Compartir con otras personas
2. Cambia a "Cualquier persona con el enlace puede ver"
3. Copia la URL del navegador y pégala arriba

Opción B — Publicar en la web:
1. Archivo → Publicar en la web
2. Selecciona la pestaña de datos → formato CSV → Publicar
3. Copia la URL generada y pégala arriba
